# Contributing

Thank you for your interest in contributing to ACP MCP Server!

For contribution guidelines, please see the main guide:

**[ACP Contributing Guide](https://github.com/acp-protocol/acp-spec/blob/main/CONTRIBUTING.md)**

## Quick Start for MCP Server Development

```bash
git clone https://github.com/acp-protocol/acp-mcp.git
cd acp-mcp
cargo build
cargo test
```

## Before Submitting

- [ ] `cargo fmt`
- [ ] `cargo clippy -- -D warnings`
- [ ] `cargo test`
